# Akikoori's Doujin Game Reviews

同人游戏评测合集

<img src="https://raw.githubusercontent.com/Akikoori/ADGR/main/Pictures/Recognition.png" alt="Recognition" style="zoom:50%;" />

其实不是游戏"评测"。我的本意是想做一系列游戏介绍，也作为给自己玩过的一些感觉至少还行的游戏做个记录

对我来说，黄游在黄之前首先是游——因为如果只是为了黄，我多达2100部的单行本/同人志更能满足我的需求——所以比起色不色，我更看重黄油作为游戏的完成度

然而，欠佳的游戏性有时也许也能由作者的巧思与才华弥补

总体来说：**独特之处>游戏性>色**

有的游戏也许不好"用"，甚至不好玩，但它们依然可能有一些，独特的，让人耳目一新的诱人亮点

当然这里也不会全是这样的游戏。说到底只是我玩过觉得还行的游戏的记录，各种名作大作也是会有的吧，大概
